## ECMWF AIFS Single forecast

![ECMWF AIFS](/assets/catalog-thumbnails/ecmwf-aifs-single-forecast.jpg)

The AI weather model from the European Centre has entered the catalog. ECMWF's [AIFS Single](https://dynamical.org/catalog/ecmwf-aifs-single-forecast/) — 15-day global forecasts at 0.25°, 6-hourly steps, live updating. This is the single-trace (deterministic-ish?) run of ECMWF's machine learning forecast system.

## NOAA MRMS

![NOAA MRMS](/assets/catalog-thumbnails/noaa-mrms-conus-analysis-hourly.jpg)

New CONUS RADAR dataset! [MRMS](https://dynamical.org/catalog/noaa-mrms-conus-analysis-hourly/) is Multi-Radar Multi-Sensor — NOAA's merged radar and gauge precipitation analysis over CONUS, hourly, back to late 2014. If you've ever wanted to answer "how much did it actually rain here, at this exact hour" without downloading 47 files from 3 different FTP servers... this is for you.

- [Docs](https://dynamical.org/catalog/noaa-mrms-conus-analysis-hourly/)

## dynamical-catalog python library

We're testing a simple Python library: [dynamical-catalog](https://github.com/dynamical-org/dynamical-catalog). The idea is to make discovery and loading even simpler — browse the catalog, pick a dataset, and get an xarray Dataset or Store back without copy-pasting URLs. We'll handle picking the best Store backend. It's early days. If you try it, let us know what breaks.
